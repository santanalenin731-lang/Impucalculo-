Guía de Compra y Configuración de Dominio

Este documento detalla los pasos para adquirir el dominio impucalculo.com y vincularlo a tu proyecto.

IMPORTANTE: No se han realizado cambios en tu aplicación real. Esta es solo una guía informativa.

1. Dónde y Cómo Comprar el Dominio

Necesitas un Registrador de Dominios. Opciones recomendadas:

  • Namecheap: Precios competitivos y privacidad gratis.
  • Porkbun: Interfaz muy limpia.
  • GoDaddy: Popular pero cuidado con renovaciones.
  • Squarespace (antes Google Domains): Fácil integración.

Pasos Generales:

  1. Ve al sitio web del registrador (ej. namecheap.com).
  2. Busca impucalculo.com.
  3. Si está disponible, agrégalo al carrito.
  4. Completa el registro y pago.

2. Configuración Técnica (DNS)

En el panel de tu registrador (sección DNS), crea estos registros:

A. Registros "A" (Apuntan a GitHub)

Crea 4 registros tipo A con nombre @ hacia estas IPs:

  • 185.199.108.153
  • 185.199.109.153
  • 185.199.110.153
  • 185.199.111.153

B. Registro "CNAME" (Subdominio www)

Crea un registro tipo CNAME:

  • Type: CNAME
  • Host: www
  • Value: santanalenin731-lang.github.io
Nota: Los cambios DNS pueden tardar hasta 48 horas en propagarse.

3. Configuración en GitHub (Paso Final)

  1. Ve a tu repositorio en GitHub > Settings > Pages.
  2. En Custom domain, escribe: impucalculo.com.
  3. Haz clic en Save.
  4. Marca la casilla Enforce HTTPS.

¿Qué sucede automáticamene?

GitHub creará un archivo CNAME. Gracias al script de auto-sync que te instalé, este archivo se guardará y subirá solo a tu repositorio.
